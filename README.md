# Data Analysis - R

Projet d'analyse de données appliqué à des datasets réels (pollution ozone, données médicales, sondages).

## Description
Analyse statistique et modélisation de données pour identifier des relations entre variables et valider des hypothèses.

## Contenu
* Analyse de distributions (normale, binomiale, Poisson, Student, Chi²)
* Régression linéaire et analyse de corrélations
* Tests statistiques (Student, proportions, Chi²) et intervalles de confiance
* Visualisations (graphiques, histogrammes, boxplots)
* Simulations Monte Carlo

## Technologies utilisées
* R : Programmation statistique et analyse de données

## Compétences
Analyse de données • Modélisation statistique • Visualisation • R
